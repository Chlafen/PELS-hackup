### Running Locally
#### Prerequisites
* Node.js
* npm
#### Installation
1. Clone the repo

```sh
git clone
```

2. Install dependencies

```sh
pnpm install
```

3. Start the backend server

```sh
pnpm run -F backend start:dev
```

4. Start the frontend server

```sh
pnpm run -F frontend dev
```
